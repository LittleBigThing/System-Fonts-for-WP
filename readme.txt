== Description ==

This plugin adds a collection of system font stacks to WordPress. After plugin activation, the fonts become available in the Font Library as a collection of ‘Local Fonts’.
These fonts are the lightest and fastest fonts since they don’t need to be downloaded on page load.

The currently available font stacks are:

* Sans Serif
* Serif
* Monospace

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **Local Fonts**.
3. Install and activate the plugin.

= Manual installation =

1. Upload the entire `system-fonts-for-wp` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Sytem Fonts for WP plugin.

= After activation =

The fonts are available as a collection ‘Local Fonts’ in the Font Library.
